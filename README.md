# Fishway

A modern e-commerce platform for buying and selling fresh fish, built with Next.js and TypeScript. Dedicated for software engineering course in an undergraduate Computer Science program.

## Features

- **Authentication**: Login and signup pages for users
- **Buyer Dashboard**: Browse products, manage cart, and checkout
- **Seller Dashboard**: Manage products and orders
- **Product Listings**: View detailed product information
- **Responsive Design**: Optimized for mobile and desktop

## Tech Stack

- **Frontend**: Next.js 14, React, TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Heroicons (via SVG)
- **Build Tool**: Next.js CLI

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd fishway
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open** [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/`: Next.js app router pages and layouts
- `components/`: Reusable UI components
- `lib/`: Utility functions and data types
- `public/`: Static assets

## Data Types and Constraints

Currently, the app uses TypeScript data types to enforce constraints and structure for products, users, orders, etc. This ensures type safety and consistency across the application.

## Future Plans

In the future, this project will integrate Supabase as the backend for authentication, database management, and real-time features. The current data types will serve as the foundation for the database schema.
